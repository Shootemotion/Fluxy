# Fluxy

Proyecto de gestión de finanzas personales.

## Tecnologías
- Next.js
- Supabase
- Tailwind CSS

## Instalación

```bash
npm install
```

## Configuración

1. Copia el archivo `.env.example` a `.env.local`:
   ```bash
   cp .env.example .env.local
   ```
2. Completa las variables `NEXT_PUBLIC_SUPABASE_URL` y `NEXT_PUBLIC_SUPABASE_ANON_KEY` con tus credenciales de Supabase.

## Desarrollo

```bash
npm run dev
```
